VIS.JS介绍

基于浏览器的动态可视化库。该库旨在易于使用，处理大量动态数据，以及实现对数据的操作和交互。 该库由组件DataSet，Timeline，Network，Graph2d和Graph3d组成。

本篇主要将network；

Network介绍

network是一种用于将包含点和线的网络和网络之间的可视化展示。可视化易于使用，并支持自定义形状、样式、颜色、大小、图像等。网络可视化可以在任何现代浏览器上流畅地工作， 最多支持几千个节点和边。为了处理更多的节点，Network提供了集群支持。网络使用HTML画布进行渲染。
从4.0开始，网络由处理网络特定部分的各个模块组成。这些模块有自己的文档、选项、方法和事件，您可以通过单击下面列表中的模块来访问它们。

【JAVASCRIPT】

1）创建一个nodes数组
var nodes = new vis.DataSet([ {id: 1, label: 'Node 1'}, {id: 2, label: 'Node 2'}, {id: 3, label: 'Node 3'}]);
2）创建一个edges数组
var edges = new vis.DataSet([ {from: 1, to: 3},{from: 1, to: 2}]);
3）创建network实例
var container = document.getElementById('mynetwork');
4）创建data对象
var data = { nodes: nodes, edges: edges }
5）创建option
var options = { }
6）最后
var network = new vis.Network(container, data, options);
