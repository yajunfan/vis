Network中包含的公共方法有哪些

1.全局方法
destroy()

从DOM中删除network并删除所有Hammer绑定和引用,销毁该实例
return:none
network.destory()

setData({nodes:vis.DataSet[nodeary],edges:vis.DataSet[edgeary]})

对network上的所有数据的格式化，如果物理模块中启用了稳定，则network将再次稳定。首次初始化network时也会执行此方法
return:none
var nodes = new vis.DataSet ([ {id: 1, label: 'Node 1'}, {id: 2, label: 'Node 2'}, {id: 3, label: 'Node 3'}]);

setOptions(Object)

设置option,这是全局的修改options的配置，不针对与某一个节点，作用于整个network
return:none
network.setOptions(option)
eg：

on(String event name, Function callback)
eg:network.on("doubleClick", function (params) { console.log("doubleClick",params) }) 设置事件监听器;
具体的事件名称和回调函数看下表
off(String event name, Function callback)
eg:network.off("doubleClick");删除事件监听器，在次代码之前的该事件取消执行，如果没有提供事件名称，如'doubleClick',那就是将所有的network上绑定的事件取消
once(String event name, Function callback)
eg:network.once("click",function(params){});以click事件为利，只执行第一次，之后不再执行。发生之后，事件监听器将被删除
2.与canvas相关的方法
canvasToDOM({x:Number,y:Number})

此函数将画布坐标转换为DOM上的坐标。输入和输出的形式为{x：Number，y：Number}。 DOM值是相对于网络容器的
return:Object
var obj=network.canvasToDOM({x:itempos.x,y:itempos.y});

DOMtoCanvas({x:Number,y:Number})

此函数将DOM坐标转换为画布上的坐标。输入和输出的形式为{x：Number，y：Number}。 DOM值是相对于网络容器的
return:Object
var obj=network.DOMtoCanvas({x:itempos.x,y:itempos.y});

redraw()
重绘network，执行此事件的时候，会一同执行以下三个事件，我们在以下三个事件中做需要的功能逻辑
return:none
network.redraw()

同时要执行以下三件事

network.on("initRedraw", function () {console.log(3)});
network.on("beforeDrawing", function (ctx) {console.log(4)});
network.on("afterDrawing", function (ctx) {console.log(5)});

其中ctx为canvas的context对象（network生成的canvas），比如
var c=document.getElementById("myCanvas");
var cxt=c.getContext("2d");
setSize(String width, String height)
eg:network.setSize('1200','600');;
canvas的宽高

4.Layout方法
getSeed()

当节点中没有x，y，network生成的随机的节点分布，会有一个seed值，我们可以通过该方法获取到。当下次希望仍然以这个布局展示的话，可以在layout.randomSeed中设置
return:Number
var seed = network.getSeed();获取seed值
layout.randomSeed = seed;设置seed值
